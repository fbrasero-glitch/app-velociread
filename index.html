Dado que iOS restringe la ejecución de Javascript en archivos locales abiertos directamente desde la app de Archivos, tienes estas dos alternativas sencillas:

  1. Usar la App gratuita "Documents de Readdle":
    • Descarga Documents de Readdle en el App Store.
    • Pásate este archivo index.html al iPhone (por AirDrop, Email, iCloud, etc.).
    • Ábrelo dentro de la app Documents. Su visor web integrado ejecuta Javascript sin limitaciones locales, permitiendo que la app funcione offline.
  2. Guardar como App de Pantalla de Inicio (PWA):
    • Si tienes el archivo subido a una dirección web gratuita (como GitHub Pages o Vercel), ábrelo una vez en Safari.
    • Pulsa el botón Compartir en Safari y selecciona Añadir a pantalla de inicio.
    • Se creará un acceso directo que funciona como app nativa offline y con un rendimiento optimizado.
